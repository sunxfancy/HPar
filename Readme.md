HPar
====


HPar is a paralleled HTML5 parser based on Jsoup. It has 2.2x speedup in mulit-core systems compared with Jsoup.

# Build

HPar can be built by maven, in JDK 1.8 version.

```
git clone https://github.com/sunxfancy/HPar

mvn build
```
